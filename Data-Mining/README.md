# 📂 ITI Data Mining Course Portfolio 

This repository showcases a comprehensive Data Mining journey during the **ITI Intensive Program**. It covers the implementation of Association Rules, Unsupervised Learning (Clustering), and Supervised Learning (Classification) using real-world datasets.

---

## 🛠️ Projects Strategic Overview

| Project | Learning Type | Core Algorithm | Technical Focus |
| :--- | :--- | :--- | :--- |
| **01-Market-Basket-Analysis** | Association | `Apriori` | Support, Confidence, Lift|
| **02-Customer-Segmentation** | Unsupervised | `K-Means` | Elbow Method|
| **03-Churn-Prediction-App** | Supervised | `Naive Bayes` | Probabilistic Prediction & Deployment |

---

## 📝 Featured Projects

### 🛒 1. Market Basket Analysis
* **Goal:** Extract purchasing patterns from Egyptian grocery transaction data.
* **Technical Highlights:** Used the **Apriori** algorithm to generate frequent itemsets and applied association rules to identify strong relationships between products based on **Lift** and **Leverage**.
* **Tools:** `mlxtend`, `pandas`.

### 👥 2. Customer Segmentation (Capstone)
* **Goal:** Segment mall customers into distinct groups to improve targeted marketing strategies.
* **Technical Highlights:** Employed the **Elbow Method** (analyzing WCSS) to find the optimal number of clusters. Visualized customer personas using `Seaborn` pairplots.
* **Tools:** `Scikit-learn`, `Matplotlib`, `Seaborn`.

### 📊 3. Customer Churn Prediction App
* **Goal:** Build a complete end-to-end system to predict customer attrition based on demographics (Age, Tenure, Gender).
* **Technical Highlights:** Evaluated multiple **Naive Bayes** variants (`Gaussian`, `Multinomial`, `Bernoulli`). Developed a real-time web interface for model inference.
* **Tools:** `Streamlit`, `Scikit-learn`, `Joblib`.
