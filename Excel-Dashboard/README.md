# AdventureWorks Sales Performance Dashboard

## Overview
An interactive Excel dashboard analyzing AdventureWorks sales data across multiple tables.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- KPI Cards

## Key Insights
- 💰 Total Revenue: $109M+ across 121,317 orders
- 🚲 Bikes drive 86% of revenue
- 🌍 Southwest is the top market
- 📊 On-Time Ship Rate: 99.97% (only 28 delayed orders out of 121,317)
- ⚠️ 73% of shipments depend on a single carrier — a concentration risk
